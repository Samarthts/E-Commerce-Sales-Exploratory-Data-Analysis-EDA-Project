# E-Commerce-Sales-Exploratory-Data-Analysis-EDA-Project
This project demonstrates a comprehensive exploratory data analysis using a sample e-commerce sales dataset. It covers dataset structure, data quality, distributions, correlations, and business insights with Python and visualization libraries.

## Sections Covered
- Dataset Overview
- Data Quality Assessment
- Univariate Analysis
- Correlation Analysis
- Key Insights

## Tools Used
- Python 3, Pandas, Matplotlib, Seaborn
